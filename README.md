# Restaurant API

Backend API built with Express.js for managing restaurant menu items.

## Features

- Full CRUD operations
- Custom request logging middleware
- Input validation using express-validator
- Proper HTTP status codes

## Endpoints

GET /api/menu  
GET /api/menu/:id  
POST /api/menu  
PUT /api/menu/:id  
DELETE /api/menu/:id  

## Status Codes

200 - Successful GET, PUT, DELETE  
201 - Successful POST  
400 - Validation error  
404 - Not found
